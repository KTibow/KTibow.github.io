HTML

Basic HTML template (2 space indent. Please don't get mad if anything's wrong.)

What is it? This is all you need to get started. All of your head's code goes in between the two <head> and </head> tags.

All of your body's code goes in between the two <hbody> and </body> tags.

Get started with a <h1> tag, which means a header tag. Put your text after, and then close it with a </h1> tag. Put it in the body to make a Hello World!

<!DOCTYPE html>
<html>
  <head>
    <title>My Title</title>
  </head>
  <body>
  </body>
</html>

Inline JS

What is it? This lets you execute inline JS.

<script>
alert("HI!");
</script>

Include JS (place at bottom of HTML if possible, order matters though.)

What is it? This lets you include and execute JS (JavaScript) from another source. Note the type="text/javascript" is optional, but you should use it just in case.

<script src="/myscript.js" type="text/javascript"></script>

Async JS (better than plain, prefer defer if possible.)

What is it? This tag is basically the same as the previous one. With the previous one, though, the browser stops rendering the whole page, downloads the JS, executes the JS, then renders the rest of it. With this one, it downloads the JS while it's rendering, and then executes the JS when it finishes. This is default if you use JS to make a JS tag. Note that it's impossible to apply either async or defer to inline JS.

<script async src="/myscript.js" type="text/javascript"></script>

Defer JS (waits long time!)

What is it? This is like Async JS, but instead of executing it when it finishes downloading, it executes it when the page finishes rendering. Important notes: 1. defer can't apply to inline JS. 2. Don't count on it; it's better to use onload if it's critical that it works in older browsers.

<script defer src="/myscript.js" type="text/javascript"></script>

Inline CSS

What is it? This lets you use inline CSS.

<style>
* {
  font-family: "Comic Sans MS";
}
</style>

Include CSS

What is it? This lets you use include and use CSS (Cascading Style Sheets) from another source. Note that the type="text/css" is optional, but you should use it just in case.

<link rel="stylesheet" href="/mystyle.css" type="text/css" />

Defered include CSS

What is it? This lets you defer using CSS until page load. Note you may want to provide a noscript fallback. Also note this works inline.

<link rel="stylesheet" type="text/css" href="/mystyle.css" media="none" onload="if(media!='all')media='all'">
