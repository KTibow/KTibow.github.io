Installation

As soon as you click the link, it will say "This kind of file can harm your device. Do you want to keep Library.apk anyway?" But don't worry, because my app won't hurt a fly. So press "Keep/Yes".

Now, as soon as you open the file to install the app, it'll say "For security reasons, your device is not allowed to install apps from this source." In order to install the app, click the button that says "Settings", and find the app you opened the file with. Turn on the switch that says "Allow from this source", and press the back button until you get out of settings.

I don't really know what'll happen next, but just keep clicking the buttons that let you install the app. You do trust me, right?
